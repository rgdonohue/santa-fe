# Santa Fe Field Notes

A small, honest body of work about Santa Fe that combines maps, analysis, and lived experience.

## Quick Start

1. **Set up Python environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```

2. **Read the plan:**
   See `PLAN_CORE.md` for the core strategy, prioritization rules, and v0.1 outcomes.

3. **Start exploring:**
   - Check `docs/data_sources.md` for dataset links
   - Begin with `notebooks/00_exploratory/001_who_lives_where.ipynb`

## Structure

- `data/` - Raw and processed datasets
- `notebooks/` - Exploratory and analysis notebooks
- `src/` - Reusable Python modules
- `maps/` - Static map outputs
- `stories/` - Field notes (drafts and published)
- `docs/` - Documentation and metadata
